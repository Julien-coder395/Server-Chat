### Outil d'installation ###

1) Dézipper le dossier

2) Ouvrir une console pouvant exécuter python.

Pour Windows:
3) Pour démarrer le serveur se rendre avec la console dans le dossier Server-Chat/Serveur

4) Lancer la commande : python .\server_multi_thread.py

5) Pour démarrer un client se rendre avec la console dans le dossier Server-Chat/Client 

6) Lancer la commande : python .\client_tcp.py
(Exécuter cette commande autant de fois que vous souhaitez créer un client)

7) Exécuter la commande #Help pour avoir plus d'information (client et serveur).

Pour éteindre le serveur : 
1) Saissir la commande #Exit
2) Fermer la console du serveur.
